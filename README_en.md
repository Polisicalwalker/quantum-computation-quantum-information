# Nielsen & Chuang — Quantum Computation and Quantum Information

A collection of exercise solutions, organized by chapter.

## Directory Structure

```
.
├── README.md
├── README_en.md
├── chapter02/              # Chapter 2: Introduction to Quantum Mechanics
│   ├── textbook_chap2.pdf              # Textbook chapter PDF
│   ├── solutions_chap2_A4.pdf          # Solutions PDF (A4 format)
│   ├── solutions_chap2_us_letter.pdf   # Solutions PDF (US Letter format)
│   └── solutions.md                    # Solutions (Markdown)
├── chapter03/              # Chapter 3: (to be added)
└── trash/                  # Intermediate / process files
    └── chapter2_process_files/
```

## Naming Convention

| File Type | Naming Format | Example |
|-----------|--------------|---------|
| Textbook chapter | `textbook_chapXX.pdf` | `chapter02/textbook_chap2.pdf` |
| Solutions (Markdown) | `solutions.md` | `chapter02/solutions.md` |
| Solutions PDF (A4) | `solutions_chapXX_A4.pdf` | `chapter02/solutions_chap2_A4.pdf` |
| Solutions PDF (US Letter) | `solutions_chapXX_us_letter.pdf` | `chapter02/solutions_chap2_us_letter.pdf` |

## Notes

- `trash/` contains intermediate files generated during the agent workflow (extracted text, raw data, etc.). They are kept but moved out of the main directory.
- When adding a new chapter, create a `chapterXX/` folder in the root directory and follow the same naming convention.
